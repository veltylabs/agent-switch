# Fix agent-switch: tinywasm/fmt + tinywasm/mcp + tinywasm/orm API migration

## Status

**The module compiles and tests pass today** using `tinywasm/mcp v0.0.4` and `tinywasm/fmt v0.18.18`.  
Three layers need updating: the generated ORM code, the MCP registration layer, and the tests.

## Issues Found

### 1. `model_orm.go` — intermediate fmt API

The file uses `fmt.Field` but with an **older intermediate struct layout** (fmt v0.18.x):

```go
// CURRENT (wrong for v0.22.2)
{Name: "id", Type: fmt.FieldText, PK: true}

// CORRECT (v0.22.2)
{Name: "id", Type: fmt.FieldText, DB: &fmt.FieldDB{PK: true}}
```

Additionally:
- Uses `FormName() string` → must become `ModelName() string`
- `ReadAll` callbacks use `orm.Model` → must become `fmt.Model`

### 2. `mcp.go` — mcp v0.0.4 API (two versions behind)

| Current (v0.0.4) | Target (v0.1.1) |
|------------------|-----------------|
| `GetMCPToolsMetadata() []mcp.ToolMetadata` | `Tools() []mcp.Tool` |
| `mcp.ToolMetadata{Name, Description, Parameters, Execute}` | `mcp.Tool{Name, Description, InputSchema, Resource, Action, Execute}` |
| `mcp.ParameterMetadata{Name, Description, Required, Type}` | removed — replaced by `InputSchema string` (JSON) |
| `Execute: func(ctx context.Context, args map[string]any) (any, error)` | `Execute: func(ctx *context.Context, req mcp.Request) (*mcp.Result, error)` |
| `*mcp.MCPServer` | `*mcp.Server` |
| `srv.RegisterProvider(m)` | pass `m` as `ToolProvider` to `mcp.NewServer(config, providers)` |

### 3. `mcp_test.go` — calls removed types

- `m.GetMCPToolsMetadata()` → `m.Tools()`
- `mcp.ToolMetadata` / `mcp.ParameterMetadata` struct access → updated field names
- `TestRegisterTools` if present → update to `mcp.NewServer` pattern

## Stage 3 — Rewrite `mcp.go`

**File**: `mcp.go`

### New mcp.Tool struct (v0.1.1)

```go
type Tool struct {
    Name        string
    Description string
    InputSchema string // JSON schema (optional; omit if no required params)
    Resource    string // required — RBAC resource identifier
    Action      byte   // required — 'c', 'r', 'u', 'd'
    Execute     func(ctx *context.Context, req Request) (*Result, error)
}
```

Handler arguments arrive as a JSON string in `req.Params.Arguments`. Decode with `json.Unmarshal`:

```go
var args struct {
    IsEnabled bool   `json:"is_enabled"`
    ChangedBy string `json:"changed_by"`
    Reason    string `json:"reason"`
}
json.Unmarshal([]byte(req.Params.Arguments), &args)
```

Result helpers:
```go
mcp.Text(string)                              // success
&mcp.Result{IsError: true, Content: "msg"}   // error
```

### Complete rewrite

Replace the entire `mcp.go`:

```go
//go:build !wasm

package agentswitch

import (
    "encoding/json"

    "github.com/tinywasm/context"
    "github.com/tinywasm/fmt"
    "github.com/tinywasm/mcp"
    "github.com/tinywasm/orm"
    "github.com/tinywasm/unixid"
)

type Module struct {
    db  *orm.DB
    uid *unixid.UnixID
}

func New(db *orm.DB) (*Module, error) {
    if err := db.CreateTable(&AgentSwitch{}); err != nil {
        return nil, err
    }
    u, err := unixid.NewUnixID()
    if err != nil {
        return nil, err
    }
    return &Module{db: db, uid: u}, nil
}

// Tools implements mcp.ToolProvider.
func (m *Module) Tools() []mcp.Tool {
    return []mcp.Tool{
        {
            Name:        "get_agent_status",
            Description: "Returns the current agent enabled/disabled status.",
            Resource:    "agent_switch",
            Action:      'r',
            Execute:     m.getStatus,
        },
        {
            Name:        "toggle_agent_status",
            Description: "Enables or disables the agent. Append-only audit log.",
            Resource:    "agent_switch",
            Action:      'u',
            Execute:     m.toggle,
        },
    }
}

func (m *Module) getStatus(ctx *context.Context, req mcp.Request) (*mcp.Result, error) {
    rows, err := ReadAllAgentSwitch(
        m.db.Query(&AgentSwitch{}).OrderBy(AgentSwitch_.ID).Desc().Limit(1),
    )
    if err != nil {
        return &mcp.Result{IsError: true, Content: fmt.Err("database", "unavailable").Error()}, nil
    }
    if len(rows) == 0 {
        b, _ := json.Marshal(map[string]any{"is_enabled": false, "changed_at": int64(0)})
        return mcp.Text(string(b)), nil
    }
    r := rows[0]
    ts, _, err := m.uid.Parse(r.ID)
    if err != nil {
        return &mcp.Result{IsError: true, Content: err.Error()}, nil
    }
    b, _ := json.Marshal(map[string]any{
        "is_enabled": r.IsEnabled,
        "changed_by": r.ChangedBy,
        "changed_at": ts,
        "reason":     r.Reason,
    })
    return mcp.Text(string(b)), nil
}

func (m *Module) toggle(ctx *context.Context, req mcp.Request) (*mcp.Result, error) {
    var args struct {
        IsEnabled *bool  `json:"is_enabled"`
        ChangedBy string `json:"changed_by"`
        Reason    string `json:"reason"`
    }
    if err := json.Unmarshal([]byte(req.Params.Arguments), &args); err != nil || args.IsEnabled == nil {
        return &mcp.Result{IsError: true, Content: fmt.Err("params", "invalid").Error()}, nil
    }
    if args.ChangedBy == "" {
        return &mcp.Result{IsError: true, Content: fmt.Err("params", "invalid").Error()}, nil
    }
    row := &AgentSwitch{
        ID:        m.uid.GetNewID(),
        IsEnabled: *args.IsEnabled,
        ChangedBy: args.ChangedBy,
        Reason:    args.Reason,
    }
    if err := m.db.Create(row); err != nil {
        return &mcp.Result{IsError: true, Content: fmt.Err("database", "unavailable").Error()}, nil
    }
    b, _ := json.Marshal(map[string]any{"ok": true, "is_enabled": *args.IsEnabled})
    return mcp.Text(string(b)), nil
}
```

> **Note on `ctx.Std()`**: the handler receives `*context.Context` from `github.com/tinywasm/context`. If this module ever needs to pass a stdlib context downstream, check whether `tinywasm/context` exposes a `.Std()` conversion. Currently neither `getStatus` nor `toggle` pass context to other functions, so no conversion is needed.

> **Note on `mcp.Request.Params.Arguments`**: in the new API, `Arguments` is a `string` (raw JSON), not `map[string]any`. Use `json.Unmarshal([]byte(req.Params.Arguments), &target)` to decode.
